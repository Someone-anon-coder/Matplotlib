ion and ioff function of the matplotlib.pyplot module
This function is used to enable and disable interactive mode in plotting

This function is useful to show live data changes with matplotlib visualization

Example: 
import matplotlib.pyplot as plt
import numpy as np

plt.ion()

x = np.linspace(0, 10, 100)
for i in range(1, 4):
    plt.plot(x, np.sin(i * x))
    plt.title(f'Interactive Plot - Wave {i}')

    plt.pause(1)
    plt.savefig(f'Plots/interactive_plot_{i}.png')

    plt.clf()
plt.ioff()

Output: 
 Interactive Plot - Wave 1
 Interactive Plot - Wave 2
 Interactive Plot - Wave 3

This program plots sin waves with different frequencies and is shown live in interactive mode
This program saves the figure and clears the plot
